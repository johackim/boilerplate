My personal Next.js boilerplate

## Installation

```bash
# Clone this repository
git clone https://github.com/johackim/boilerplate

# Go into the repository
cd boilerplate

# Install dependencies
yarn install

# Run the app
npm run dev

# Go to http://localhost:3000
```

## Running the tests

```bash
npm test
```
